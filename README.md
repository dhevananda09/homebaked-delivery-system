HomeBaked is a full-stack web application that allows customers to browse bakery items, add products to cart, checkout, and receive order confirmation.

## Problem Statement
Small and medium bakeries often use manual order systems which leads to delays, miscommunication, and poor customer experience.  
HomeBaked solves this by providing a digital online ordering platform.

## Tech Stack
- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MongoDB
- APIs: REST API

## Features
- Browse bakery products with details and pricing
- Add/remove items from cart
- Checkout with customer details
- Generates unique Order ID
- Stores order details in MongoDB
- Order confirmation page
